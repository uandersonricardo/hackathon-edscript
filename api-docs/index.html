    Overview¶

    This documentation is organized into two main sections:

    References
    Provides a description of the API endpoints available to users.

    How To
    A guide outlining the steps (primarily the sequence of API requests) needed to achieve a specific goal or state.
    The How To section is divided into two parts: endpoints that require login and those that don`t. Both sections include example requests.

    Notice for Firefox Users
    When accessing the local HTML documentation in Firefox:
    - Clicking a link normally functions as expected.
    - Using Ctrl+Click (to open a link in a new tab) may not navigate to the intended section of the page.
    This behavior is specific to Firefox; Chrome and Safari did not exhibit this issue.
