    Retrieve Odds - Markets and Groups¶

    Markets and groups define the context of the bet (e.g., Match Result, Total Goals, Handicap).

    GET /api-v2/bet-type-groups/{device}/{language}/{trader}
    Retrieves available markets and their configuration.

    Example response snippet:

  "success": true,
  "responseCodes": [],
  "data": [

    betTypeGroups¶

    • A betTypeGroup represents a betting market, i.e., a category of betting selections that are related.
    • Fields in the example:
      • betTypeGroupId - unique identifier for this market.
      • betTypeGroupName - human-readable name describing the market (e.g., "1x2", "Handicap", "Total games").
      • mainBtgName - the main, overarching category this market belongs to.

    Interpretation: betTypeGroups are betting markets, containing multiple betting selections (betTypes) that users can place.

    betTypeGroupMarkets¶

    • A betTypeGroupMarket is a collection of related betting markets, organized by context or event stage.
    • Fields in the example:
      • betTypeGroupMarketName - the human-readable market name, like "Match", "Games", "Sets", "Goals", "Handicap".
      • betTypeGroups - the list of betting markets in that market.
    • In the example, the "Match" market contains betting markets like "Winner (incl. super over)".

    Interpretation: betTypeGroupMarkets are categories of betting markets, often reflecting how bets are grouped in the UI.

    mainBtgName¶

    • mainBtgName identifies the primary category under which a betting market falls.
    • It groups similar betting markets across multiple betTypeGroupMarkets. For example, several markets may contain bets under the main category "Winner".`

    Interpretation: mainBtgName is a logical classification of betting markets, useful for UI grouping and analytics.

    Relationships Between Them¶

    Layer Example Description
    Main Bet Type Group (mainBtgName) "Winner" Broad classification of a bet, used across multiple markets.
    Market (betTypeGroupMarketName) "Match", "Sets", "Goals" Groups related bets for a specific event context.
    Bet Type Group (betTypeGroupName) "Winner (incl. super over)", "Total games" Actual bet options that a user can place.

    Key insight: Despite the name betType, it does not refer to a low-level technical type—it is really the betting option the user can select.
